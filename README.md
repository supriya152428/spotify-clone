# Spotify Clone

A mini Spotify player built with HTML, CSS, and JavaScript. Play your favorite songs with an interactive and responsive interface.

### Features
- Play, pause, and skip tracks
- Progress bar to see and control song timing
- Song list with covers and names
- Responsive and clean design
- Small animation while music plays

### Technologies
- HTML – structure of the player
- CSS – styling and layout
- JavaScript – audio controls and interactivity
